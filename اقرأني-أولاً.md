# مشروع تجارتي — نظام إدارة المبيعات والمخزون

يتكوّن المشروع من مجلدين:

## 1. برنامج_المبيعات/ — المنتج

برنامج سطح المكتب المتكامل (Python + SQLite). يعمل بدون إنترنت.

التشغيل السريع:

```bash
cd برنامج_المبيعات
pip install -r requirements.txt
python main.py
```

الدخول: `admin` / `admin`.

التفاصيل في `برنامج_المبيعات/README.md` و`دليل_المستخدم.md`.
لبناء نسخة `.exe` للعملاء: شغّل `build_exe.bat`.

## 2. الموقع/ — الموقع التسويقي

صفحة هبوط عربية لبيع البرنامج. افتح `الموقع/index.html` في المتصفح.
التفاصيل في `الموقع/README.md`.

## ما الذي أُنجز

- برنامج كامل: نقطة بيع (POS)، مخزون وباركود، فواتير، عملاء وموردون، مشتريات،
  تقارير، لوحة تحكم، إعدادات وصلاحيات.
- واجهة عربية بالكامل (RTL) بتصميم داكن عصري.
- قاعدة بيانات محلية مع بيانات تجريبية جاهزة للعرض.
- طباعة الفواتير وتصدير التقارير إلى Excel.
- موقع تسويقي متجاوب مع الجوال.
- توثيق كامل وسكربت بناء exe.

## خطوات يدوية متبقية (اختيارية)

- تخصيص أرقام التواصل والأسعار في الموقع (مذكورة في `الموقع/README.md`).
- ربط نموذج الطلب في الموقع بخدمة بريد (Formspree مثلاً).
- بناء ملف `.exe` على جهازك عبر `build_exe.bat` قبل التسليم للعميل.

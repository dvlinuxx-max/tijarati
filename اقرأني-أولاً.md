# 📦 مشروع «تجارتي» — نظام إدارة المبيعات والمخزون

هذا المشروع يتكوّن من **مجلدين**:

## 1️⃣ `برنامج_المبيعات/` — المنتج الذي تبيعه
برنامج سطح المكتب المتكامل (بايثون + SQLite). يعمل بدون إنترنت.

**للتشغيل السريع:**
```bash
cd برنامج_المبيعات
pip install -r requirements.txt
python main.py
```
الدخول: `admin` / `admin`

📄 اقرأ `برنامج_المبيعات/README.md` و`دليل_المستخدم.md` للتفاصيل الكاملة.
🏗️ لبناء نسخة `.exe` للعملاء: شغّل `build_exe.bat`.

## 2️⃣ `الموقع/` — الموقع التسويقي لبيع البرنامج
صفحة هبوط عربية احترافية. افتح `الموقع/index.html` في المتصفح.

📄 اقرأ `الموقع/README.md` لمعرفة كيفية تخصيص الأرقام والأسعار والنشر.

## ✅ ما الذي أُنجز؟
- ✅ برنامج كامل: نقطة بيع (POS)، مخزون وباركود، فواتير، عملاء وموردين، مشتريات، تقارير، لوحة تحكم، إعدادات وصلاحيات.
- ✅ واجهة عربية بالكامل (RTL) بتصميم عصري داكن.
- ✅ قاعدة بيانات محلية مع بيانات تجريبية جاهزة للعرض.
- ✅ طباعة الفواتير + تصدير التقارير إلى Excel.
- ✅ موقع تسويقي متجاوب مع الجوال.
- ✅ توثيق كامل وسكربت بناء exe.

## 🛠️ خطوات يدوية متبقية (اختيارية)
- تخصيص أرقام التواصل والأسعار في الموقع (مكتوبة في `الموقع/README.md`).
- ربط نموذج الطلب في الموقع بخدمة بريد (Formspree مثلاً).
- بناء ملف `.exe` على جهازك عبر `build_exe.bat` قبل التسليم للعميل.

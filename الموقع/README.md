# 🌐 الموقع التسويقي — برنامج تجارتي

صفحة هبوط (Landing Page) عربية احترافية لعرض وبيع برنامج «تجارتي».

## التشغيل
افتح ملف `index.html` مباشرة في المتصفح — لا يحتاج أي خادم أو تثبيت.

أو شغّل خادماً محلياً للمعاينة:
```bash
python -m http.server 8000
# ثم افتح: http://localhost:8000
```

## المحتوى
- **index.html** — كل أقسام الصفحة (البطل، المميزات، اللقطات، الأسعار، الأسئلة، التواصل).
- **assets/style.css** — التصميم (داكن عصري، RTL، متجاوب مع الجوال).
- **assets/script.js** — قائمة الجوال، النموذج، وتأثيرات الظهور.

## قبل النشر — خصّص التالي:
1. **أرقام التواصل**: ابحث في `index.html` عن `9647700000000` و`sales@tijarati.app` واستبدلها بأرقامك.
2. **الأسعار**: عدّل قسم `#pricing` بالأسعار التي تناسبك.
3. **ربط النموذج**: حالياً النموذج عرض توضيحي. لربطه فعلياً استخدم خدمة مثل
   [Formspree](https://formspree.io) أو [Web3Forms](https://web3forms.com)
   (مجانية) — فقط ضع رابط الخدمة في `action` الخاص بالـ `<form>`.

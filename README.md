# تجارتي

نظام إدارة مبيعات ومخزون لسطح المكتب، موجّه للمحلات والشركات الصغيرة والمتوسطة.
يعمل بدون إنترنت، وبياناته محلية في قاعدة SQLite. الواجهة عربية بالكامل (RTL).

المستودع يضم جزأين:

- `برنامج_المبيعات/` — تطبيق سطح المكتب (Python + Tkinter + SQLite).
- `الموقع/` — صفحة هبوط تسويقية ثابتة (HTML/CSS/JS) للتعريف بالبرنامج.

## البرنامج

نقطة بيع وباركود، إدارة مخزون، فواتير، عملاء وموردون مع متابعة الديون، مشتريات،
لوحة تحكم وتقارير قابلة للتصدير إلى Excel، ومستخدمون بصلاحيات (مدير/كاشير).

### التشغيل من المصدر

```bash
cd برنامج_المبيعات
pip install -r requirements.txt
python main.py
```

الدخول التجريبي: `admin` / `admin`. عند أول تشغيل تُنشأ قاعدة بيانات تجريبية في
`برنامج_المبيعات/data/tijarati.db`؛ احذف الملف للبدء بقاعدة فارغة.

### بناء نسخة تنفيذية

```bash
pip install pyinstaller
cd برنامج_المبيعات
build_exe.bat
```

الناتج في `برنامج_المبيعات/dist/`.

## البنية

```
برنامج_المبيعات/
  main.py              نقطة التشغيل
  app/
    config.py          الإعدادات والألوان ومساعد العربية RTL
    database.py        قاعدة البيانات ومنطق العمل
    login.py           شاشة الدخول
    main_window.py     النافذة الرئيسية والتنقل
    widgets.py         عناصر واجهة مشتركة
    pages/             الوحدات: POS، فواتير، مخزون، عملاء، موردون، تقارير
  data/                قاعدة البيانات (تُنشأ تلقائياً)
  build_exe.bat        سكربت بناء exe
الموقع/
  index.html
  assets/
db_manager.py          أداة مساعدة لإدارة قاعدة البيانات
```

## ملاحظات

- البيانات محلية بالكامل، لا تُرسل إلى أي خادم.
- كلمات المرور مخزّنة كـ SHA-256.
- خذ نسخة احتياطية من `data/tijarati.db` دورياً.

تفاصيل الاستخدام للعميل في `برنامج_المبيعات/دليل_المستخدم.md`.

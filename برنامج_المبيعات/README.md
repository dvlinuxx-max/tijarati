# 🧾 تجارتي — نظام إدارة المبيعات والمخزون المتكامل

برنامج سطح مكتب عربي متكامل لإدارة المبيعات والمخزون للمحلات والشركات الصغيرة والمتوسطة.
يعمل بدون إنترنت، وبياناته محفوظة محلياً على جهازك (قاعدة بيانات SQLite).

---

## ✨ المميزات

- **نقطة بيع (POS) سريعة** — بحث بالاسم أو مسح الباركود، سلة، خصومات، وطباعة فورية للفاتورة.
- **إدارة المخزون** — أصناف، تصنيفات، باركود، أسعار التكلفة والبيع، وتنبيهات نفاد المخزون.
- **الفواتير** — سجل كامل لكل المبيعات مع إعادة الطباعة والبحث.
- **العملاء والموردون** — سجل كامل مع متابعة الديون والمستحقات وتسديدها.
- **المشتريات** — تسجيل فواتير الشراء من الموردين (تزيد المخزون تلقائياً).
- **لوحة تحكم** — إحصائيات لحظية ورسم بياني لمبيعات آخر 14 يوم وأفضل المنتجات.
- **التقارير** — تحليل المبيعات والأرباح حسب الفترة + تصدير إلى Excel/CSV.
- **المستخدمون والصلاحيات** — مدير عام / مدير / كاشير، لكل دور صلاحياته.
- **تخصيص كامل** — اسم الشركة، الشعار النصي، العملة، الضريبة، وملاحظات الفاتورة.
- **واجهة عربية بالكامل** (RTL) بتصميم عصري داكن مريح للعين.

## 🚀 التشغيل (للمطوّر / النسخة المصدرية)

```bash
# 1) تثبيت المتطلبات
pip install -r requirements.txt

# 2) تشغيل البرنامج
python main.py
```

### بيانات الدخول التجريبية
| المستخدم | كلمة المرور | الصلاحية |
|---------|------------|---------|
| `admin` | `admin`    | مدير عام (كل الصلاحيات) |
| `cashier` | `1234`   | كاشير (بيع فقط) |

> 💡 عند أول تشغيل تُنشأ قاعدة بيانات تجريبية فيها منتجات وعملاء وفواتير لعرض إمكانيات البرنامج.
> يمكنك حذف ملف `data/tijarati.db` للبدء بقاعدة بيانات فارغة.

---

## 📦 بناء نسخة تنفيذية (.exe) للتوزيع على العملاء

البرنامج يُوزَّع كملف `.exe` واحد لا يحتاج العميل لتثبيت بايثون.

```bash
pip install pyinstaller
build_exe.bat
```

ستجد الملف التنفيذي داخل مجلد `dist/`. راجع ملف **build_exe.bat** للتفاصيل.

## 📁 بنية المشروع

```
برنامج_المبيعات/
├── main.py                 ← نقطة التشغيل
├── requirements.txt
├── build_exe.bat           ← سكربت بناء نسخة exe
├── دليل_المستخدم.md         ← دليل الاستخدام للعميل
├── data/                   ← قاعدة البيانات (تُنشأ تلقائياً)
└── app/
    ├── config.py           ← الإعدادات والألوان ومساعد العربية RTL
    ├── database.py         ← قاعدة البيانات ومنطق العمل
    ├── widgets.py          ← عناصر الواجهة المشتركة
    ├── login.py            ← شاشة الدخول
    ├── main_window.py      ← النافذة الرئيسية والتنقل
    └── pages/              ← وحدات البرنامج (POS، فواتير، مخزون، تقارير...)
```
